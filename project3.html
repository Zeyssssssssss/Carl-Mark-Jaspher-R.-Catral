Fina's Dormitory Management System

A Comprehensive Platform for Efficient Dormitory Operations

Project Overview

Fina's Dormitory Management System is a user-friendly platform that simplifies dormitory operations. The system enables efficient management of tenants, room assignments, payments, and maintenance requests. It includes an automated reminder system for payments and provides an intuitive tenant portal for account management and tracking. The system helps reduce administrative workload, minimize errors, and provides valuable insights for better decision-making.

Key Features:

  • Tenant Registration & Room Assignment
  • Online Payment Tracking & History
  • Automated Payment Reminders & Notifications
  • Maintenance Request Management
  • Tenant Portal for Account Management
  • Admin Dashboard for Monitoring & Reporting
  • Mobile-friendly Interface for Easy Access

Technologies Used:

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: PHP
  • Database: MySQL
  • Version Control: Git, GitHub
